# INSTALL

> npm install

Alternatively, if you have yarn installed,

> yarn install

# RUNNING LOCALLY

To Run,

> npm run dev

or

> yarn dev

# DOCKER

To make it go on Docker,

> docker built -t febp .
> docker run -p 8080:80 --name=febp febp
